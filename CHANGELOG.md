# 变更记录

## [0.1.0] - 2026-04-01

### 初始化
- 项目初始化，建立前后端分离架构
- 前端：Vue3 + TypeScript + Element Plus + Vite
- 后端：Node.js + Express
- Docker支持：docker-compose编排

### 功能
- 统一门户框架（侧边栏导航、布局）
- 全局资源视图（Dashboard）
  - 统计卡片展示
  - 资源拓扑图（ECharts力导向图）
  - 四大超算中心信息展示
- 登录页面
- 各模块占位页面（函数服务、工作流、监控、数学库）

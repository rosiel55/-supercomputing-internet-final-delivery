# 算力数据可视化大屏

## 项目概述
多数据中心算力资源调度与任务可视化大屏系统。

## 技术栈
- **前端**: Vue3 + TypeScript + Vite
- **图表**: ECharts + D3.js
- **3D可视化**: Three.js
- **后端**: Node.js (待开发)

## 项目结构
```
├── frontend/          # 前端项目
├── backend/           # 后端服务 (待开发)
├── docs/              # 文档
└── README.md
```
